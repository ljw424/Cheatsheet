HTML Table Reference

Input Tags

Tag Name Description
<input type="button"> Button A simple push button.
<input type="checkbox"> Checkbox A box that is ticked when activated.
<input type="date"> Date An input field that allows the user to enter the date.
<input type="datetime-local"> Date and Time An input field that allows the user to enter the date and the time.
<input type="email"> Email An input field that allows the user to enter and edit an email address.
<input type="file"> File Allows the user to upload a file from their device.
<input type="image"> Graphical Button Creates a graphical submit button (a button taking the form of an image rather than text).
<input type="radio"> Radio Button Small buttons which are filled or highlighted when selected.
<input type="search"> Search Field Text fields which allow the user to enter search queries.
<input type="url"> File Allows the user to enter and edit a url.
